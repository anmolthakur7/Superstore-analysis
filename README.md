# Superstore Sales Analysis

## About
This project analyzes the Sample Superstore dataset to find correlations
between Discount and Profit, and profit distribution by product category.

## Files
- `Project__2_.ipynb` — Main analysis notebook
- `SampleSuperstore.csv` — Dataset
- `report_python_2.docx` — Project report

## How to Run
Open in Google Colab or run locally with Jupyter Notebook.
